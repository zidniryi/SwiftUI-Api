# My Food

An iOS application made with SwiftUI and Xcode Version 12.x


# Usage
Open **GetApiFood.xcworkspace**  with your Xcode



# Technologies & Tools

 - SwiftUI
 - Alamofire
- Maps
- Combine
- URLImage
- Xcode 12.x
- Cocoapods
